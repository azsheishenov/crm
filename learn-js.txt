// localStorage

localStorage - Это локальное хранилище в браузере у пользователя с 
которым мы можем взаимодействовать, хранить данные в localStorage мы можем только в 
формате json т.е строки. Объем хранилища состоявляет в зависимости от браузера в котором 
используется от 5-10 мб.

localStorage.getItem('key') - метод который позволяет по ключу получить сохраненное 
значение из локального браузерного хранилища пользователя

localStorage.setItem('key', value) - метод который позволяет сохранить в локальное хранилище 
пользователя небольшие данные

localStorage.removeItem('key') - метод который позволяет удалить пару ключ-значние 
по названию ключа 

localStorage.clear() - метод полностью очищает localStorage

// sessionStorage 

sessionStorage - работает аналогично localStorage, но очищается автоматически в случае если 
пользователь закроет выкладку или браузер
